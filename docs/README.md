## 🚀 프로젝트 개요

mem0를 활용하여 사용자의 장기 기억을 효율적으로 저장하고 관리하는 시스템을 구축합니다.

### 핵심 기능
- 자동 메모리 추출 및 저장
- 수동 메모리 입력 및 자동 분류
- 지능형 검색 및 컨텍스트 활용
- Web UI를 통한 편리한 인터페이스

### 기술 스택
- Memory: mem0
- LLM: Ollama (로컬)
- Vector DB: Qdrant/ChromaDB
- Backend: FastAPI
- Frontend: React/Streamlit
